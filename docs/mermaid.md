## mermaid
Markdown上でUML図を作成できる

Github，Qiitaでもサポートされている

VSCodeの拡張機能がある 
- Markdown Preview Mermaid Support (`bpruitt-goddard.mermaid-markdown-syntax-highlighting`)
- Mermaid Markdown Syntax Highlighting (`bierner.markdown-mermaid`)

### サンプル

```mermaid
    pie
    "A": 40
    "B": 30
    "C": 20
    "D": 10
```
